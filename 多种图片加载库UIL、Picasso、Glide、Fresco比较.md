<b>共同点：</b>
1. 支持网络图片与本地存储图片加载
2. 支持内存、本地存储、网络三级加载策略，自动缓存管理及资源回收复用
3. 支持Android原生ImageView
4. 支持占位图

<b>不同点：</b>
Picasso:
1. Debug指示器：通过在图片左上角添加不同颜色的小三角来指示图片加载来源：红色（网络）、蓝色（本地存储）、绿色（内存）
2. 简单易用的流式API调用
3. 支持图片加载失败时显示的图片
4. 网络通信使用OkHttp
5. 最低支持Android API 14（Android 4.0）
6. 支持图片缩放等变换
7. 源码只有几百K
8. 如果程序代码进行混淆，需添加OkHttp混淆规则
